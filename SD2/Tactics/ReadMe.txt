How to Use
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

1. Double click on Run_Win32/Tactics.exe.
2. To play in local mode, press enter when "Play" is selected.
3. To play in networked mode, use the arrow keys to select "Host", and then press enter.
	- You will also need a client in this mode. Open the Developer Console using '~' and execute the "clone_process" command to open a new instance.
	- Once the new instance opens, open the Developer Console there and type "join <ipaddress>", where <ipaddress> is the address displayed on the host's screen.
	- This can also be done by running two instances on separate machines that are connected to the same local network.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Gameplay
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
There are two opposing factions, red and blue, each starting with one knight, mage and archer, each.
The goal is to defeat the opposing faction in this turn based game.
When it is a given actor's turn, they can Move, Wait or execute class-specific actions (like Bow for archers and Heal for mages).
Each action (other than Wait) has allowed tiles where the action can be performed.
